What are Conditional Statements?

Conditional statements are used to perform different actions based on different conditions.

JavaScript has the following conditional statements:

  • if - executes code if a condition is true
  • else - executes code if the same condition is false
  • else if - specifies a new condition to test if the first condition is false
  • switch - selects one of many code blocks to execute
  • ternary operator - shorthand for simple if/else statements

Code Examples

If/Else Statement

let age = 18;

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

Else If Statement

let time = 14;

if (time < 12) {
    console.log("Good morning!");
} else if (time < 18) {
    console.log("Good afternoon!");
} else {
    console.log("Good evening!");
}

Switch Statement

let day = 3;
let dayName;

switch (day) {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    // ... more cases ...
    default:
        dayName = "Unknown day";
}
